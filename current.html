JUNE GAME: Focus Challenge Test Your Reflexes!

Concept

A simple game where the user must click a shape (e.g., a circle or square) as soon as it appears. Measures reaction time, encourages concentration, and can be repeated to track improvement.

Features

  • Click as fast as possible when shape appears

  • Random delay before showing the shape (25 seconds)

  • Displays user's reaction time in milliseconds

  • Option to retry and compare results

  • Calm design (soft colors, minimal animations)

How to Build It (HTML/JS Version)

Heres a simple version you can copy/paste onto your website:

html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Focus Challenge</title>
  <style>
    body {
      text-align: center;
      font-family: Arial, sans-serif;
      background: #f0f9ff;
      padding-top: 50px;
    }
    #box {
      width: 100px;
      height: 100px;
      background: #4f46e5;
      margin: 50px auto;
      display: none;
      border-radius: 50%;
      cursor: pointer;
    }
    button {
      padding: 10px 20px;
      background: #10b981;
      color: white;
      border: none;
      border-radius: 8px;
      font-size: 16px;
      cursor: pointer;
    }
  </style>
</head>
<body>
  <h1> Focus Challenge</h1>
  <p>Click the circle as soon as it appears!</p>
  <div id="box"></div>
  <p id="result"></p>
  <button onclick="startGame()">Start</button>

  <script>
    let startTime, timeout;

    function showBox() {
      document.getElementById("box").style.display = "block";
      startTime = new Date().getTime();
    }

    function startGame() {
      document.getElementById("box").style.display = "none";
      document.getElementById("result").textContent = "Get ready...";
      const delay = Math.random() * 3000 + 2000; // 25 sec
      timeout = setTimeout(showBox, delay);
    }

    document.getElementById("box").onclick = function () {
      const endTime = new Date().getTime();
      const reactionTime = (endTime - startTime) / 1000;
      document.getElementById("result").textContent = `Your reaction time: ${reactionTime.toFixed(3)} seconds!`;
      this.style.display = "none";
      clearTimeout(timeout);
    };
  </script>
</body>
</html>

  • Add a Try Again button or a scoreboard to enhance engagement.
